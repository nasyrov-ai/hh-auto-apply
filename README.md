# HH Auto Apply — Автоматизация откликов на hh.ru с Claude AI

Автоматически откликается на вакансии и отвечает в чаты от твоего имени.
Каждое письмо генерирует Claude Opus — под конкретную вакансию.

## Что делает

- 📨 Рассылает отклики с персональным сопроводительным письмом
- 🤖 Письмо пишет Claude Opus — анализирует вакансию и твоё резюме
- 💬 Отвечает в чаты работодателям (только когда они написали)
- 📈 Поднимает резюме в поиске каждый день (бесплатно)
- ⏰ Работает по расписанию через cron — руки не нужны

## Быстрый старт

```bash
# 1. Установить зависимости
pip install hh-applicant-tool playwright
playwright install chromium

# 2. Авторизоваться (откроется браузер)
hh-applicant-tool authorize --no-headless --manual

# 3. Запустить
./run.sh
```

## Структура

```
hh-auto-apply/
├── run.sh          # главный скрипт (запускает всё)
├── config.py       # твой профиль и промпты
├── prompts/
│   ├── cover.txt   # промпт для сопроводительных писем
│   └── chat.txt    # промпт для ответов в чаты
└── logs/           # логи запусков
```

## Как работает Claude

Клод получает:
- Описание вакансии
- Твоё резюме (название + навыки)
- Инструкции из промпта

И пишет живое, конкретное письмо — без шаблонов.

## Стек

- [hh-applicant-tool](https://github.com/s3rgeym/hh-applicant-tool) — работа с hh.ru API
- [Claude Code CLI](https://claude.ai/code) — генерация писем
- Python 3.11+, bash
